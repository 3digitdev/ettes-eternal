# Advancing Skills
### Session Experience
At the end of every _gaming session_, all players gain **5 Experience Points (XP)** that must be spent on advancing their Skills. **You cannot bank XP between sessions; spend it immediately**.  There's not really a reason to bank them anyways.
## Skill Advancement Costs
Each time you buy a Skill Advancement, you increase that Skill by +1.  You must pay a cost according to the **new level of that Skill**:
_e.g. If you are Advancing a Skill from 65-66, you would pay 3 XP for this, rather than 2 XP._

| NEW Skill Level | Cost/level |
|:---------------:|:----------:|
|      11-40      |    1 XP    |
|      41-65      |    2 XP    |
|      66-80      |    3 XP    |
See [[Skills#Skill Limits|Skill Limits]] for details on what limits there are to advancing Skills.

*Math note:  The astute will notice that once multiple skills hit 65, you will only be able to advance ONE of those skills per session, because you don't earn enough XP in a session to pay for two (need 6 XP, but only gain 5 XP per session).  **This is by design!** Tradeoffs are important.*
#### Skill Advancement Limit
You can never Advance a Skill more than once per session.  You can purchase as many Skill Advancements as you have XP for, but you could not (for example) level your 20 Insight to 25 using all 5 XP from the most recent session.

# Class Level Progression
Although there are many ways to determine how/when players should level up, this system takes an opinionated approach to the problem so as to provide a balanced and consistent progression to the game, allowing GMs not to worry about keeping player levels up or various fiddly numbers and balancing and edge cases.
### Recommended Class Level Timeline
*See [[#System Scaling|Guidelines]] below for information on deviating from this timeline!*

| Level | Sessions to Next Level | Level on Session... | Campaign Milestone                    |

| 1     | 1                      | 1                   |                                       |
| 2     | 1                      | 2                   |                                       |
| 3     | 2                      | 3                   |                                       |
| 4     | 2                      | 5                   |                                       |
| 5     | 2                      | 7                   | T2 Perks; Class Path; Journeyman Tier |
| 6     | 2                      | 9                   |                                       |
| 7     | 3                      | 11                  |                                       |
| 8     | 3                      | 14                  |                                       |
| 9     | 3                      | 17                  |                                       |
| 10    | 3                      | 20                  | T3 Perks; Expert Tier                 |
| 11    | 4                      | 23                  |                                       |
| 12    | 4                      | 27                  |                                       |
| 13    | 4                      | 31                  |                                       |
| 14    | 4                      | 35                  | Master Tier                           |

_**Total Campaign Length:** ~64 sessions + however many spent at Level 20; coming out to ~1 year, 3 months assuming consistent weekly sessions._

# Perk Acquisition
Every character receives 1 Perk for free regardless of class at every new Class Level (including Level 1), for a total of **20 Perks by level 20** _(including the free Level 1 Perk)_

You start the game only able to buy Tier 1 Perks.  As shown in the table above, other Perks unlock at later levels:
- **Tier 1:** Available at all levels
- **Tier 2:** Available starting Level 5
- **Tier 3:** Available starting Level 10
- **Tier 4:** Available starting Level 15

# GM Guidelines
## Session Productivity
Sometimes your players just faff around for 3 hours at the table; sitting in a tavern gambling, or a session-long shopping trip.  In these scenarios, it is recommended that you **should not include these sessions in determining advancement**.
- **Award the players 0 XP for the session**
- **Session does not count toward next level**

**VERY IMPORTANT:** **Both conditions above** must be applied to prevent **power imbalance** and gaming the system.  If for example you give XP but don't count the session, the balance between expected skill level maximums and current level get thrown off, and the game will feel less balanced as a result.
## System Scaling
**Important:** To modify campaign length, modify the XP rewards AND level timing in equal but opposite factors:
- **Faster Campaign:** 2x XP per session (10 XP) + half the sessions between levels = ~32 session campaign
- **Slower Campaign:** 0.5x XP per session (2-3 XP per session; alternate) + double sessions between levels = ~128 session campaign
**Warning:** Tweaking only one part of the system will create power level imbalances.

# Example Progression
## Skill Advancement Timeline
_Assuming you're starting at 35 for the Skill:_
- **To reach 40:** 5 sessions minimum (5 XP total; Level 4+)
- **To reach 65:** 30 sessions minimum (55 XP total; Level 12+)
- **To max at 80:** 45 sessions minimum (100 XP total; Level 16+)
## Resource Trade-offs
Players maxing one skill (35→80) over 45 sessions have 125 XP remaining for other skills, creating meaningful opportunity costs for specialization vs. breadth.
