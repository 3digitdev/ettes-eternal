# Advancing Skills
## Session Experience
At the end of every _[[GM Guidelines#Session Productivity|productive]] gaming session_, all players gain **Experience Points (XP)** based on what **Class Tier** they currently are that are be spent on advancing their Skills. **You can bank XP between sessions to save up for new advancements**
## Tier Advancement Reward
Additionally, you also gain +10 flat XP when you increase your Class Tier.

The following table captures all of these benefits:

| Class Tier | XP per Session | Sessions | Tier Reward | Total XP |
| ---------- | :------------: | :------: | :---------: | :------: |
| Apprentice |      2 XP      |    7     |      0      |    14    |
| Journeyman |      3 XP      |    14    |     10      |    52    |
| Expert     |      4 XP      |    16    |     10      |    74    |
| Master     |      5 XP      |    15    |     10      |    85    |
| Legendary  |      6 XP      |    12    |     10      |    82    |
## Skill Advancement Costs
Each time you buy a Skill Advancement, you increase that Skill by +1.  You must pay a cost according to the **new level of that Skill**:
_e.g. If you are Advancing a Skill from 65-66, you would pay 3 XP for this, rather than 2 XP._

| Skill Level | Cost per Advance |
| :---------: | :--------------: |
|    31-40    |       1 XP       |
|    41-55    |       2 XP       |
|    56-70    |       4 XP       |
|    71-80    |       6 XP       |
|    81-90    |       8 XP       |

See [[Skills#Skill Limits|Skill Limits]] for details on what limits there are to advancing Skills.

# Class Level Progression
Although there are many ways to determine how/when players should level up, Ettes Eternal takes an opinionated approach to the problem so as to provide a balanced and consistent progression to the game, allowing GMs not to worry about keeping player levels up or various fiddly numbers and balancing and edge cases.
### Recommended Class Level Timeline
*See [[#System Scaling|Guidelines]] below for information on deviating from this timeline!*

| Level | Sessions to Next Level | Level after Session... |     Campaign Milestone      |

| 1     |           1            |           1            |                             |
| 2     |           1            |           2            |                             |
| 3     |           2            |           3            |                             |
| 4     |           2            |           5            |                             |
| 5     |           2            |           7            | Journeyman Tier; Class Path |
| 6     |           2            |           9            |                             |
| 7     |           3            |           11           |                             |
| 8     |           3            |           14           |                             |
| 9     |           3            |           17           |                             |
| 10    |           3            |           20           |         Expert Tier         |
| 11    |           4            |           23           |                             |
| 12    |           4            |           27           |                             |
| 13    |           4            |           31           |                             |
| 14    |           4            |           35           |         Master Tier         |
| 15    |           5            |           39           |                             |

_**Total Campaign Length:** ~64 sessions + however many spent at Level 20; coming out to ~1 year, 3 months assuming consistent weekly sessions._

# Perk Acquisition
Every character receives 1 Perk for free regardless of class at every new Class Level (including Level 1), for a total of **20 Perks by level 20**

You start the game only able to buy Apprentice Tier Perks.  As shown in the table above, higher Tier Perks unlock at later levels along with every other Tiered System.
