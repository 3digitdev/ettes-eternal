## Illusory Sound
**Level:** Novice  
**Mana Cost:** 0  
**Range:** 10 hexes  
**Target:** Within range  
**Duration:** Instant

**Effect:** Create minor auditory illusions and sound effects at target location.

|Class Tier|Additional Effect|
|---|---|
|Apprentice|Simple sounds (voices, animal noises, objects)|
|Journeyman|Simple sounds, can throw voice to different location|
|Expert|Complex sounds, can mimic specific voices heard before|
|Master|Complex sounds, can create multiple simultaneous sounds|
|Legendary|Perfect mimicry, can create conversations between multiple voices|

## Mage Hand
**Level:** Novice  
**Mana Cost:** 0  
**Range:** 6 hexes  
**Target:** One small object  
**Duration:** Sustained

**Effect:** Telekinetically manipulate small objects at range.

**Critical Effect:** Duration becomes 5 minutes instead of Sustained (no concentration required).

| Class Tier | Weight Limit                                                     |
| ---------- | ---------------------------------------------------------------- |
| Apprentice | 5 lbs; Basic manipulation                                        |
| Journeyman | 10 lbs; Basic manipulation                                       |
| Expert     | 15 lbs; Can manipulate multiple small objects up to weight limit |
| Master     | 20 lbs; Can perform fine manipulation (writing, lockpicking)     |
| Legendary  | 30 lbs; Can exert significant force (opening heavy doors)        |
