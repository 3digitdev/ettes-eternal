# Class Skill Bundles
If you would like some help with picking out Class Skills, Ettes Eternal introduces an option called "Class Skill Bundles" which are a set of ~5 pre-defined Class Skills for each.  These Skills represent your class's primary proficiencies, matching to the archetype each represents.

**Additionally**, every you pick 3 other Skills not on your Class's list, and those 3 skills become Class Skills as well, for a total of **8 Class Skills** like normal.
### Class Skill Bundle Table

| :----------: | :-------------: | :--------------------: | :-----------------: | :-----------------------: | :----------------: |
|     Bard     | Melee OR Ranged | Influence OR Deception | Knowledge (Pick 1)  |          Insight          |      Agility       |
|  Barbarian*  |      Melee      |       Athletics        |      Endurance      |        Perception         | Survival OR Nature |
|    Druid     |  Spellcasting   |   Nature OR Survival   |     Perception      |         Medicine          |     Willpower      |
|   Fighter*   |      Melee      |         Ranged         |      Athletics      |         Endurance         |      Agility       |
|   Hunter*    |     Ranged      |       Perception       |       Agility       |    Survival OR Nature     |      Crafting      |
|     Monk     |      Melee      |  Agility OR Athletics  |       Insight       |         Willpower         |     Endurance      |
|    Rogue*    |    Thievery     |        Agility         | Ranged OR <br>Melee |        Perception         |     Deception      |
|  Sorcerer*   |  Spellcasting   |         Arcana         |      Willpower      |   Insight OR Influence    |      Agility       |
|    Witch     |  Spellcasting   |    Arcana OR Nature    |       Insight       |         Medicine          |     Deception      |
| Necromancer❌ |  Spellcasting   |   Arcana OR Religion   |      Influence      |         Medicine          |     Willpower      |
